School Management API

Welcome to the School Management API. This API allows you to manage school data by adding new schools and retrieving a list of schools based on proximity to a user-specified location.

Available Endpoints

  • POST /addSchool - Add a new school by providing the name, address, latitude, and longitude.
  • GET /listSchools?latitude=-1.9473&longitude=30.0503 - Retrieve a list of schools sorted by proximity to a given latitude and longitude.

Technologies Used

  • Node.js
  • Express.js
  • MySQL

For more information, please visit the GitHub repository.
